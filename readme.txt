Hermes is messenger, make the user registration method more diverse.

== Description ==

The plugin uses Tencent Cloud SMS to make the user registration method more diverse.
The SMS notification function for registering and modifying personal information is currently implemented.

== Installation ==

= From your WordPress dashboard =

1. Visit 'Plugins > Add New'
2. Search for 'Hermes'
3. Activate Hermes from your Plugins page.
4. Visit 'Settings > Hermes' and adjust your configuration.

= From WordPress.org =

1. Download Hermes.
2. Upload the 'hermes' directory to your '/wp-content/plugins/' directory.
3. Activate Hermes from your Plugins page.
4. Visit 'Settings > Hermes' and adjust your configuration.
